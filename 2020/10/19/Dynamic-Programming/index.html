Dynamic Programming

Posted on 2020-10-19 In Algorithms

This blog is the learning note for the dynamic programming in the course algorithm and complexity analysis given by Prof. Gao Xiaofeng@SJTU

Introduction

  • 贪心算法： 通过每一步得到一个局部最优解，一步一步地得到一个解。
  • 分治法： 将大问题划分为同类型且相互独立的小问题， 再将子问题的解合并形成大问题的解。
  • 动态规划： 将一个问题分为重叠的小问题，然后逐步合并

Applications:

  • Bioinformatics
  • Unix diff for comparing two files
  • Viterbi for hidden Markov models

Introduction Example: Weighted Interval Scheduling

工作$j$开始于$s_j$并且结束于$f_j$.每项任务有自己的权重$w_j >0$. 任何两项不重叠的工作是可以兼容的.

目标: 找到权重和最大的子集,使得子集内任务互相兼容.

贪心算法仅适用于当所有工作的权重都为1时,如果有了weight,等于变成了两个维度.

解决:

首先将所有工作按照结束时间从小到大排序. 定义$p(j)$为小于$j$ 的最大工作序号.
